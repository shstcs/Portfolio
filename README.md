# Portfolio
게임 개발자를 꿈꾸고 달려가는 성연호의 포트폴리오입니다.

## Intro
**캡스톤 프로젝트**로 처음 시작했던 유니티 프로젝트부터,  
**내일배움캠프**와 함께 진행했던 4개월간의 여러 프로젝트들을 정리하였습니다.  
어떻게 진행되었고, 얼마나 배우고 성장했는지 봐주시면 감사하겠습니다.

## Projects

팀원 간의 협업에 대하여 배울 수 있었습니다.
